# TNG Design System

A modern, accessible React component library with built-in theming, dark mode, and multiple accent colors. Zero external runtime dependencies â€” just React.

## Quick Start

```bash
npm install @ninoop/tng-design-system
```

```tsx
import { ThemeProvider, Button } from "@ninoop/tng-design-system";
import "@ninoop/tng-design-system/theme.css";

function App() {
  return (
    <ThemeProvider defaultTheme="light" defaultAccent="blue">
      <Button variant="solid">Click me</Button>
    </ThemeProvider>
  );
}
```

## Theming

Use the **Theme** and **Accent** toolbar controls above to preview every component in light/dark mode with any accent color.

```tsx
import { ThemeProvider, useTheme } from "@ninoop/tng-design-system";

// Wrap your app
<ThemeProvider defaultTheme="system" defaultAccent="purple">
  <App />
</ThemeProvider>

// Access theme anywhere
function MyComponent() {
  const { theme, resolvedTheme, accentColor, setTheme, setAccentColor } = useTheme();
  return <button onClick={() => setTheme("dark")}>Go dark</button>;
}
```

## Design Tokens

All components use CSS custom properties from `theme.css`. Override any token in your app:

```css
:root {
  --ui-radius-md: 6px;    /* smaller rounded corners */
  --ui-font: "Inter", sans-serif; /* different font */
}
```
